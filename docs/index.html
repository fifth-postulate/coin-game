Coin Game

A present for Linde's 30th birthday

The Coin Game is played by two players, taking turns. Each turn a player takes a number of coins from a single row.

In the misere(☹) game the player that takes the last coin loses. In the normal(☺) game the player wins when she takes the last coin.

An analysis

We will analyse the game in order to tell which player will win, and how that player should play.

Known Positions

We will first analyse some position with great regularity.

One coin per row

When each row had only one coin, the only option a player has is to take a coin from a row. Therefor the number of rows, determines the number of turns that is played. This in turn determines the winner.

This proofs the following theorem.

For a position in which each row has a single coin, the following holds.

  • The first player has a winning strategy when the number of rows is even.
  • The second player has a winning strategy when the number of rows is odd.

An example of a position with a winning strategy for the second player.

One row of coins

When there is only one row of coins and there are at least two coins, the first player has a winning strategy. For she can take all but the last coin, leaving the second player with a losing position.

This proofs the following theorem.

For a position with only one row the following holds.

  • If there is only one coin the first player loses.
  • For any other number of coins the first player has a winning strategy.

An example of a position with a winning strategy for the first player.

Two rows

We will spend some time analysing all positions with two rows.

We will do this by case analyses. We will designate a position by a sequence of integers. For example the 2,3 position is demontrated below.

1,n

This is a winning position for the first player for she can play to 1.

n,n

This is a win for the second player. For what ever the first player plays she can either play to t,t or she finds herself in 1,n, which is a winning position for her.

m,n

This is a win for the first player for she can play to u,u, where u is the minimum of m and n.

A winning position for the first player.
